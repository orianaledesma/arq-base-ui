# ARQ-BASE-UI

## Installation

1. Install vscode

2. Install node v10.x (recommended)
   `nvm instal 10.16.0`

3. Install dependencies
   `npm install`

4. Run project on develop mode
   `npm run start`

## Test

-   All tests:
    `npm run test`

-   Individual test:
    `npm run test somePattern.test`

## Build

A. `npm run build`
B. `npm run build:monitor` (to open webpack monitor)

Output folder: _dist/_
